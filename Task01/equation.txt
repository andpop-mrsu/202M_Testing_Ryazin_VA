Чек-лист для решения квадратного уравнения
==========================================

1. **Два различных вещественных корня**
   - Входные данные: a = 1, b = -12, c = 1
   - Ожидаемый результат: [1.0, 11.0]
   - Описание: Обычный случай, когда у уравнения два вещественных корня.

2. **Один вещественный корень**
   - Входные данные: a = 1, b = -6, c = 9
   - Ожидаемый результат: [3.0]
   - Описание: Дискриминант равен нулю, уравнение имеет один корень.

3. **Нет вещественных корней**
   - Входные данные: a = 2, b = -4, c = 7
   - Ожидаемый результат: Нет вещественных корней
   - Описание: Дискриминант отрицательный, вещественных корней нет.

4. **Коэффициент b равен нулю**
   - Входные данные: a = 4, b = 0, c = -4
   - Ожидаемый результат: [-1.0, 1.0]
   - Описание: Квадратное уравнение без линейного члена.

5. **Коэффициент c равен нулю**
   - Входные данные: a = 5, b = -25, c = 0
   - Ожидаемый результат: [0.0, 5.0]
   - Описание: В уравнении один из корней равен нулю.

6. **Не является квадратным уравнением**
   - Входные данные: a = 0, b = 7, c = 14
   - Ожидаемый результат: Ошибка (Ошибка ввода)
   - Описание: Некорректный случай, так как уравнение перестает быть квадратным.
